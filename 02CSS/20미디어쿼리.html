미디어쿼리

미디어의 타입이 아닌, 미디어의 특성에 따라 스타일 적용함.


		width : 화면(창)의 너비 (★중요)
		height : 화면(창)의 높이
		device-width : 디바이스 너비
		device-height : 디바이스 높이
		orientation : 디바이스의 방향(가로, 세로)
		aspect-ratio : 창의 가로세로비
		device-aspect-ratio : 디바이스의 가로세로비
		color : 컬러인 경우, 해당 색의 비트수
		color-index : 출력 디바이스가 갖고 있는 색상표의 엔트리수
		monochrome : 흑백
		resolution : 해상도
